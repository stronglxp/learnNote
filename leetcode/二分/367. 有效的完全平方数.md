#### 367. 有效的完全平方数

链接：https://leetcode-cn.com/problems/valid-perfect-square/

```java
class Solution {
    public boolean isPerfectSquare(int num) {
        long left = 1, right = num;
        while (left <= right) {
            long mid = (left + right) / 2;
            if (mid * mid == num) {
                return true;
            } else if (mid * mid > num) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }

        return false;
    }
}
```

注意整数相乘不要越界即可。
